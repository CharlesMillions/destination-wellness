# Destination Wellness - Website Specification

## Project Overview
- **Project Name:** Destination Wellness
- **Type:** Single-page holistic health network website
- **Core Functionality:** Showcase a network of naturopathic doctors and holistic health advisors
- **Target Users:** People seeking holistic health practitioners, wellness services

## Visual Design

### Color Palette
- **Primary Green:** #2D5A4A (deep forest green)
- **Secondary Green:** #4A8B6F (sage green)
- **Accent:** #7BC4A8 (light mint)
- **Water Blue:** #5DADE2 (calm blue for bubbles)
- **Background:** #F8FAF9 (off-white with green tint)
- **Text Dark:** #1A3329
- **Text Light:** #FFFFFF

### Typography
- **Headings:** "Cormorant Garamond", serif (elegant, natural feel)
- **Body:** "Nunito Sans", sans-serif (clean, approachable)

### Visual Elements
- **Leaf motif:** SVG leaf shapes as decorative elements
- **Water bubbles:** Animated floating circles with blur effect
- **Organic shapes:** Rounded containers, flowing lines
- **Subtle gradients:** Green-to-mint transitions

### Animations
- Floating bubble animation (CSS keyframes)
- Leaf sway subtle animation
- Smooth scroll behavior
- Fade-in on scroll for sections

## Page Sections

### 1. Hero Section
- Large headline: "Your Journey to Wellness Starts Here"
- Subheadline about holistic health network
- Animated bubbles background
- Call-to-action button: "Find Your Health Advisor"

### 2. About Section
- Mission statement about holistic healing
- Description of the network approach
- Leafdecorative element

### 3. Our Network Section
- Grid of practitioner cards (placeholder images)
- Categories: Naturopathic Doctors, Health Advisors, Wellness Coaches
- Each card: photo placeholder, name, title, specialty

### 4. Services Section
- Holistic health services offered
- Icon-based service list with leaf accents

### 5. Contact Section
- Contact form (visual only)
- Contact information placeholder

### 6. Footer
- Copyright
- Social links placeholders

## Acceptance Criteria
- [ ] Green color scheme dominant throughout
- [ ] Leaf motifs visible in design
- [ ] Animated water bubbles in hero/background
- [ ] Network section shows practitioner cards
- [ ] Smooth animations and transitions
- [ ] Mobile responsive
- [ ] Loads without errors
